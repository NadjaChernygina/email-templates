# Email Templates

This repository contains production-ready HTML email templates built with
email client compatibility, responsiveness, and personalization in mind.

The templates are designed for marketing and CRM campaigns and follow
email best practices for cross-client rendering and deliverability.


## Features

- Table-based HTML layouts for reliable rendering across email clients
- Responsive and hybrid (fluid) design patterns
- Inline CSS styles optimized for Outlook and Gmail
- Liquid templating for dynamic personalization
- Conditional content and fallback logic for missing data
- Modular and reusable template structure
- Image assets hosted via public HTTPS URLs


## Personalization

Templates use Liquid to support dynamic content and personalization, including:

- User attributes (e.g. first name, subscription status)
- Conditional content blocks
- Dynamic article and recommendation loops
- Safe fallback handling to prevent broken layouts


## Email Client Support

Templates are tested and optimized for major email clients, including:

- Gmail (desktop & mobile)
- Outlook (desktop)
- iOS Mail / Apple Mail
- Yahoo Mail
